# Music-advisor

Персональный музыкальный консультант, который может строить предложения на основе предпочтений и делиться ссылками 
на новые выпуски и избранные плейлисты.

### В коде используются:
- Spotify API;
- HTTP Request/Response;
- Обработка JSON;
- Strategy design pattern;
- MVC design pattern;
- Gradle.
