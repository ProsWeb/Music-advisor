# Music-advisor

Персональный музыкальный консультант, который будет делать предложения на основе предпочтений и даже делится ссылками 
на новые выпуски и избранные плейлисты.

В коде используются:
Spotify API;
HTTP Request/Response;
работа с JSON;
Strategy design pattern;
Gradle.
